# 语音指令系统

一个基于Opus编解码和WebSocket的实时语音处理系统。

## 系统要求

### Windows
- JDK 11 或更高版本
- Maven 3.6 或更高版本
- CMake 3.10 或更高版本
- Visual Studio 2019 或更高版本（需要C++支持）
- vcpkg 包管理器

### macOS
- JDK 11 或更高版本
- Maven 3.6 或更高版本
- CMake 3.10 或更高版本
- Xcode Command Line Tools
- Homebrew 包管理器

### Linux
- JDK 11 或更高版本
- Maven 3.6 或更高版本
- CMake 3.10 或更高版本
- GCC/G++ 7.0 或更高版本
- pkg-config
